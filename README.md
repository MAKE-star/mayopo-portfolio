## Deploy

This is a static site — no build step required.

**Netlify:** Drag the entire folder into the Netlify deploy drop zone, or connect the GitHub repo and it deploys automatically on every push.

**GitHub Pages:** Push to a repo, go to Settings → Pages → set source to main branch / root.
